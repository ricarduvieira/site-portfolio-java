PROJETO 3: ACOMPANHAMENTO DE ENTREGAS

Esse dashboard tem como objetivo realizar o acompanhamento das entregas de uma transportadora podendo analisar:

- Quantidade de entregas

- Valor do frete

- Entregas em andamento

- Entregas atrasadas

TECNOLOGIAS DO PROJETO

POWER BI

SQL

ESTRUTURA E DESENVOLVIMENTO

A construção deste deshboard foi feita com dados que foram carregados de um banco de dados SQL Server que estava em um servidor local. Os dados foram e tranformados via power query onde foram retiradas colunas que não seriam utilizadas, também foram editadas as colunas de data que estavam em formato de texto. Posteriormente foram desenvolvidas as vizualizões que compoem os modulos do painel.
